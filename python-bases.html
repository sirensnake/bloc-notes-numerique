Python est un langage simple et puissant. Cette fiche présente ses bases : types de données, variables, conditions, boucles, et plus encore pour bien débuter.

🔢 Types de base

Python propose plusieurs types : int (entier), float (décimal), str (texte), bool (vrai/faux).

x = 5
y = 3.14
nom = "Guillaume"
est_actif = True

📦 Variables

Déclaration simple sans typage forcé :

age = 30
nom = "Python"

⚖️ Conditions

Utilisation de if, elif et else :

if age > 18:
    print("Majeur")
else:
    print("Mineur")

🔁 Boucles

Les deux boucles principales sont for et while :

for i in range(5):
    print(i)

while est_actif:
    print("Actif")
    est_actif = False
🎁 Bonus : mini-exemples pratiques

Création d'une fonction simple :

def saluer(nom):
    return f"Bonjour, {nom} !"

Et utilisation :

print(saluer("Guillaume"))
