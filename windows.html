Windows est le système d'exploitation de Microsoft qui domine le marché des PC. Cette fiche couvre les commandes essentielles, les outils d'administration et les bonnes pratiques pour gérer efficacement un système Windows.

⌨️ Invite de commandes (CMD)

Commandes essentielles en ligne de commande Windows :

ipconfig               # Affiche la configuration IP
ipconfig /all          # Affiche la config complète
ping google.com        # Teste la connectivité
tracert google.com     # Trace la route vers une destination
netstat -ano           # Affiche les connexions réseau
tasklist               # Liste les processus en cours
taskkill /IM process   # Termine un processus
dir                    # Liste les fichiers et dossiers
cd /d D:\              # Change de lecteur et répertoire
attrib +h +s fichier   # Cache un fichier ou dossier
shutdown /s /t 0       # Arrêt immédiat du système

⚙️ Gestion des services

Gestion des services Windows depuis la ligne de commande :

sc query              # Liste tous les services
sc query state= all   # Liste tous les services (actifs et inactifs)
sc start nom_service  # Démarre un service
sc stop nom_service   # Arrête un service
sc config nom_service start= auto    # Configure le démarrage automatique
sc config nom_service start= demand  # Configure le démarrage manuel
sc config nom_service start= disabled # Désactive le service
sc delete nom_service # Supprime un service (prudence !)

Via PowerShell (plus moderne) :

Get-Service            # Liste tous les services
Start-Service nom_service
Stop-Service nom_service
Set-Service nom_service -StartupType Automatic

🔑 Registre Windows

Le registre est une base de données centralisée stockant les configurations du système et des applications :

Ruches principales :

  • HKEY_LOCAL_MACHINE (HKLM) : Configuration système
  • HKEY_CURRENT_USER (HKCU) : Configuration utilisateur actuel
  • HKEY_USERS (HKU) : Configuration de tous les utilisateurs
  • HKEY_CLASSES_ROOT : Associations de fichiers
  • HKEY_CURRENT_CONFIG : Configuration matérielle actuelle

Commandes cmd pour le registre :

reg query HKLM\Software    # Interroge une clé
reg add HKLM\Software\MonApp /v Version /t REG_SZ /d "1.0" # Ajoute une valeur
reg delete HKLM\Software\MonApp /v Version # Supprime une valeur
regedit                    # Lance l'éditeur graphique

⚠️ La modification du registre peut rendre le système instable. Toujours sauvegarder avant modification.

🛡️ Sécurité et pare-feu

Configuration du pare-feu Windows :

netsh advfirewall set allprofiles state on   # Active le pare-feu
netsh advfirewall set allprofiles state off  # Désactive le pare-feu
netsh advfirewall show allprofiles           # Affiche l'état du pare-feu
netsh advfirewall firewall add rule name="SSH" dir=in action=allow protocol=TCP localport=22
netsh advfirewall firewall delete rule name="SSH"

Permissions des fichiers et dossiers :

icacls fichier /grant User:F    # Donne contrôle total à un utilisateur
icacls dossier /deny User:F     # Refuse l'accès
icacls fichier /remove User     # Supprime les permissions
icacls fichier /setowner Admin  # Change le propriétaire

📂 Emplacements importants

Dossiers système essentiels :

  • C:\Windows : Dossier système principal
  • C:\Program Files : Applications 64 bits
  • C:\Program Files (x86) : Applications 32 bits
  • C:\Users\%USERNAME% : Profil utilisateur
  • C:\Windows\System32 : Fichiers système et outils
  • C:\Windows\Temp : Fichiers temporaires système
  • %APPDATA% : Données d'application (roaming)
  • %LOCALAPPDATA% : Données d'application locale
  • %TEMP% : Fichiers temporaires utilisateur

🛠️ Outils d'administration

Outils système utiles (accessible via Win+R) :

msconfig        # Configuration du système
msinfo32        # Informations système
mmc             # Console de gestion Microsoft
compmgmt.msc    # Gestion de l'ordinateur
devmgmt.msc     # Gestionnaire de périphériques
services.msc    # Services Windows
lusrmgr.msc     # Utilisateurs et groupes locaux
gpedit.msc      # Éditeur de stratégie de groupe locale
regedit         # Éditeur du registre
perfmon         # Moniteur de performances
eventvwr        # Observateur d'événements

⌨️ Raccourcis clavier

Raccourcis Windows essentiels :

  • Win + R : Ouvre Exécuter
  • Win + E : Ouvre l'Explorateur de fichiers
  • Win + X : Menu administrateur rapide
  • Win + L : Verrouille la session
  • Alt + Tab : Basculer entre les applications
  • Alt + F4 : Ferme la fenêtre active
  • Ctrl + Shift + Esc : Gestionnaire des tâches
  • Win + I : Paramètres système
  • Win + D : Affiche le bureau
  • Win + Pause : Propriétés système
