❌ Exceptions courantes

# Division par zéro
x = 10 / 0  # ZeroDivisionError

# Accès à une variable non définie
print(y)  # NameError

# Conversion invalide
int("texte")  # ValueError

🛠️ Gestion des exceptions : try / except

try:
    x = 10 / 0
except ZeroDivisionError:
    print("Division par zéro interdite")

📌 Bloc finally

try:
    f = open("fichier.txt")
except FileNotFoundError:
    print("Fichier introuvable")
finally:
    print("Bloc finally exécuté")

⚠️ Lever une exception manuellement

raise ValueError("Valeur invalide")
