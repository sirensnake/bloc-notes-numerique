📌 Introduction à Linux

Linux est un système d'exploitation libre et open-source. Il est largement utilisé dans les serveurs, les systèmes embarqués, les ordinateurs personnels, et dans l'univers DevOps. Voici une sélection de commandes essentielles pour la ligne de commande sous Linux.

📁 Navigation

pwd

👉 Affiche le chemin absolu du répertoire courant.

ls -l

👉 Liste les fichiers avec détails (permissions, taille, date, etc.).

cd /chemin/vers/dossier

👉 Change de répertoire.

📂 Gestion de fichiers

touch fichier.txt

👉 Crée un fichier vide.

mkdir nouveau_dossier

👉 Crée un nouveau répertoire.

cp source.txt destination.txt

👉 Copie un fichier.

mv ancien.txt nouveau.txt

👉 Déplace ou renomme un fichier.

rm fichier.txt

👉 Supprime un fichier.

🔍 Recherche et aide

man ls

👉 Affiche le manuel de la commande ls.

grep "mot" fichier.txt

👉 Recherche un mot dans un fichier.
