📌 Raccourcis clavier dans le terminal

  • Ctrl + C : interrompre la commande en cours
  • Ctrl + L : nettoyer l'écran
  • Ctrl + A : aller au début de la ligne
  • Ctrl + E : aller à la fin de la ligne
  • Ctrl + R : rechercher une commande précédente

🧠 Astuces générales

!!                    # Réexécute la dernière commande
!sudo !!              # Réexécute avec sudo
history               # Historique des commandes
alias ll='ls -alF'    # Crée un alias
unalias ll            # Supprime un alias

🧹 Nettoyage

sudo apt autoremove   # Supprime les paquets inutilisés
sudo apt clean        # Vide le cache
