📊 Vue système

top                       # Vue dynamique des processus
htop                      # Version améliorée (si installée)
vmstat                    # Statistiques mémoire/CPU

⚙️ Gestion des processus

ps aux                    # Liste tous les processus
kill PID                 # Terminer un processus
top                      # Vue interactive
xkill                     # Cliquer pour tuer une fenêtre graphique

📈 Surveillance

uptime                   # Durée depuis le démarrage
watch -n 1 free -h      # Affichage mémoire RAM toutes les secondes
journalctl              # Voir les logs système (systemd)
