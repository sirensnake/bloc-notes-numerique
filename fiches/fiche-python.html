📌 Introduction à Python

Python est un langage de programmation populaire, apprécié pour sa lisibilité et sa polyvalence. Il est utilisé dans l'automatisation, l'analyse de données, l'intelligence artificielle, le développement web et bien plus.

🌟 Objectifs de cette fiche

  • Savoir exécuter un script Python
  • Gérer les packages avec pip
  • Connaître les types de données de base
  • Écrire une fonction et une boucle simple

🚀 Premiers pas

python --version

👉 Vérifie la version de Python installée.

python script.py

👉 Exécute un fichier Python.

pip install nom_du_package

👉 Installe un package Python (ex. pip install requests ).
