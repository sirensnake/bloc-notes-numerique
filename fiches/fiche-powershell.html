⚙️ Présentation

PowerShell est un langage de script et un shell développé par Microsoft, principalement utilisé pour l'automatisation des tâches système sous Windows (et désormais multiplateforme via PowerShell Core).

1. ▶️ Exécution de scripts

# Autoriser les scripts (à faire une fois)
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

# Exécuter un script
./mon_script.ps1

2. 🗂️ Commandes de base

Get-Help nom_commande     # Affiche l'aide
Get-Command               # Liste des commandes disponibles
Get-Process               # Liste des processus en cours
Get-Service               # Liste des services

3. 📁 Gestion des fichiers

Get-ChildItem             # Liste les fichiers (équivalent de ls)
Set-Location D:\          # Change de répertoire (équivalent de cd)
Copy-Item source dest     # Copie de fichier
Remove-Item fichier.txt   # Supprime un fichier

4. 🔁 Boucles et conditions

# Boucle ForEach
$liste = @("fichier1.txt", "fichier2.txt")
foreach ($f in $liste) {
  Write-Output $f
}

# Condition If
if ($a -eq 10) {
  Write-Output "Égal à 10"
} else {
  Write-Output "Différent"
}

5. 📄 Variables et tableaux

$nom = "Guillaume"
$valeurs = @(1, 2, 3, 4)
$valeurs[0]    # Affiche 1
$valeurs.Count # Nombre d’éléments

6. 💬 Fonctions

function Bonjour {
  param([string]$nom)
  Write-Output "Bonjour $nom"
}

Bonjour -nom "Guillaume"

7. 🔐 Administration système

# Liste les utilisateurs locaux
Get-LocalUser

# Créer un nouvel utilisateur (admin requis)
New-LocalUser -Name "invite" -NoPassword
