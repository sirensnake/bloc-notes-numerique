La programmation orientée objet (POO) en Python permet de structurer le code en regroupant données et comportements dans des objets. Cette fiche vous guide à travers les bases de la POO : classes, objets, constructeurs, héritage, encapsulation, etc.

🏗️ Définir une classe

class Chien:
    def aboyer(self):
        print("Ouaf !")

🔨 Instancier un objet

medor = Chien()
medor.aboyer()

🧱 Constructeur (__init__)

class Chien:
    def __init__(self, nom):
        self.nom = nom

📐 Héritage

class Animal:
    def parler(self):
        print("Je suis un animal")

class Chat(Animal):
    def parler(self):
        print("Miaou")

🔒 Encapsulation

class Compte:
    def __init__(self):
        self.__solde = 0

    def deposer(self, montant):
        self.__solde += montant
