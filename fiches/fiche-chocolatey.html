📥 Installation

Set-ExecutionPolicy Bypass -Scope Process -Force; `
[System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; `
iwr https://chocolatey.org/install.ps1 -UseBasicParsing | iex

Cette commande permet d’installer Chocolatey depuis PowerShell en contournant les restrictions d’exécution.

📦 Commandes de base

  • Installer un paquet : 
    choco install nom_du_paquet
  • Désinstaller un paquet : 
    choco uninstall nom_du_paquet
  • Mettre à jour un paquet : 
    choco upgrade nom_du_paquet
  • Mettre à jour tous les paquets : 
    choco upgrade all
  • Lister les paquets installés : 
    choco list -lo

🔍 Rechercher un paquet

choco search nom_du_paquet

⚙️ Configuration avancée

  • Modifier le cache de téléchargement : 
    choco config set cacheLocation D:\choco-cache
  • Ajouter une nouvelle source : 
    choco source add -n=nom -s="https://source-url/"
  • Voir les configurations : 
    choco config list
