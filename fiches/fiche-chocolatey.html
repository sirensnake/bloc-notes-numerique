📌 Présentation de Chocolatey

Chocolatey est un gestionnaire de paquets pour Windows. Il permet d'installer, de mettre à jour et de désinstaller facilement des logiciels depuis le terminal PowerShell ou CMD. C'est un outil puissant pour automatiser l’installation de logiciels sur une nouvelle machine ou maintenir son système à jour.

🌟 Objectifs de cette fiche

  • Installer et configurer Chocolatey
  • Utiliser les principales commandes
  • Créer des scripts d'installation groupée
  • Gérer ses logiciels de manière rapide et efficace

📥 Installation de Chocolatey

Ouvre PowerShell en tant qu’administrateur et exécute cette commande :

Set-ExecutionPolicy Bypass -Scope Process -Force; `
[System.Net.ServicePointManager]::SecurityProtocol = `
[System.Net.ServicePointManager]::SecurityProtocol -bor 3072; `
iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

📦 Commandes essentielles

Voici les commandes de base les plus utiles, accompagnées de leur description :

choco install firefox vlc 7zip -y

👉 Installe plusieurs logiciels en une seule commande. Le -y évite les confirmations manuelles.

choco upgrade all -y

👉 Met à jour tous les logiciels installés par Chocolatey vers leur dernière version.

choco uninstall notepadplusplus -y

👉 Supprime le logiciel indiqué (ici Notepad++), toujours sans confirmation avec -y .

choco list --local-only

👉 Affiche tous les logiciels installés sur la machine via Chocolatey.

choco search libreoffice

👉 Recherche un paquet par nom (ici LibreOffice) dans le catalogue de Chocolatey.

choco info git

👉 Fournit les informations détaillées sur un paquet, comme la version disponible et sa description.

choco clean

👉 Supprime les fichiers temporaires créés par Chocolatey, utile pour libérer de l’espace.

🛠️ Exemples de scripts d'installation groupée

choco install git vscode nodejs python3 -y

👉 Parfait pour une machine de développement avec Git, VSCode, Node.js et Python.

💡 Astuces pratiques

  • Crée un fichier install.bat contenant ta liste de logiciels à réinstaller facilement
  • Automatise la maintenance avec choco upgrade all -y programmé en tâche planifiée
  • Utilise choco outdated pour voir les logiciels obsolètes
