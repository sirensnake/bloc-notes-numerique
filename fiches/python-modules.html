📂 Importer un module standard

import math
print(math.sqrt(16))

🎯 Importer partiellement

from math import pi, sqrt
print(pi)
print(sqrt(25))

💼 Installer un module externe avec pip

pip install requests

Puis l'utiliser :

import requests
reponse = requests.get("https://api.github.com")
print(reponse.status_code)

🧱 Créer son propre module

# fichier monmodule.py
def saluer():
    print("Salut depuis mon module")

# dans un autre fichier
import monmodule
monmodule.saluer()
