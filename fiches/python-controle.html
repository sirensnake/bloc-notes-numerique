🔁 Instructions conditionnelles

age = 20
if age >= 18:
    print("Majeur")
else:
    print("Mineur")

🔄 Boucles for

for i in range(5):
    print(i)

♾️ Boucles while

compteur = 0
while compteur < 3:
    print(compteur)
    compteur += 1

⛔ Instructions break et continue

for i in range(5):
    if i == 3:
        break
    print(i)
for i in range(5):
    if i == 2:
        continue
    print(i)
