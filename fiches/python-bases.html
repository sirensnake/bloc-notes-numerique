📌 Présentation

Python est un langage de programmation interprété, polyvalent, facile à apprendre, largement utilisé dans les domaines de l'automatisation, du web, de l'analyse de données et de l'intelligence artificielle.

🚀 Exécuter un script

python mon_script.py

🧮 Types de base

  • int — nombres entiers
  • float — nombres à virgule flottante
  • str — chaînes de caractères
  • bool — booléens (True, False)

📚 Affectation et affichage

x = 5
y = "Bonjour"
print(x)
print(y)
