📌 Introduction à PowerShell

PowerShell est un interpréteur de commandes et un langage de script développé par Microsoft. Il est très utilisé dans l'administration système, la gestion de tâches automatisées, et le scripting sur Windows.

🔧 Commandes essentielles

Get-Command

👉 Liste toutes les commandes disponibles.

Get-Help Get-Command

👉 Fournit l’aide intégrée pour une commande donnée.

Get-Process

👉 Affiche les processus en cours.

Stop-Process -Name notepad

👉 Arrête le processus Notepad.

Get-Service

👉 Liste tous les services Windows.

Start-Service nom

👉 Démarre un service.

Set-ExecutionPolicy RemoteSigned

👉 Modifie la stratégie d’exécution pour autoriser les scripts locaux.

📜 Script de base

# Script PowerShell simple
Write-Output "Bonjour, Guillaume !"
$date = Get-Date
Write-Output "Nous sommes le $date"

👉 Ce script affiche un message et la date actuelle.
