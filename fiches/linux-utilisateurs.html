👤 Gestion des utilisateurs

adduser nom_utilisateur          # Ajouter un utilisateur
passwd nom_utilisateur          # Modifier son mot de passe
usermod -aG sudo utilisateur    # Ajouter au groupe sudo
id nom_utilisateur              # Voir les groupes d’un utilisateur

👥 Groupes et appartenance

groupadd nom_du_groupe          # Créer un groupe
gpasswd -a utilisateur groupe  # Ajouter un utilisateur à un groupe
deluser utilisateur groupe     # Retirer d’un groupe

🔐 Droits et permissions

chmod +x script.sh              # Ajouter droit d'exécution
chmod 755 fichier              # Définir permissions numériques
chown user:group fichier       # Changer propriétaire et groupe
