🌍 Interfaces et adresses IP

ip a                        # Affiche les interfaces réseau
ip link show               # Affiche les interfaces réseau (simplifié)
ip route                   # Affiche la table de routage

📡 Tests de connectivité

ping 8.8.8.8                # Test de connexion vers une IP
ping google.fr             # Test DNS + connexion
traceroute google.fr       # Affiche le chemin vers un hôte

🔎 Analyse des ports

netstat -tuln               # Ports en écoute (ancien)
ss -tuln                   # Ports en écoute (recommandé)
lsof -i                    # Liste des connexions actives
