📌 Introduction à Docker

Docker est un outil de conteneurisation permettant d'encapsuler une application avec ses dépendances dans un conteneur isolé, portable et réutilisable. Idéal pour les développeurs, DevOps et admins système.

🚀 Commandes de base

docker --version

Affiche la version de Docker.

docker ps -a

Liste tous les conteneurs, actifs ou stoppés.

docker images

Affiche les images présentes sur l'hôte.

docker run -it ubuntu bash

Démarre un conteneur Ubuntu avec terminal interactif.

docker stop <id>

Arrête un conteneur.

docker rm <id>

Supprime un conteneur.

📦 Dockerfile de base

FROM python:3.10
WORKDIR /app
COPY . .
RUN pip install -r requirements.txt
CMD ["python", "app.py"]

Utilise docker build -t mon-app . pour créer l'image.

🛠️ Docker Compose

version: '3'
services:
  web:
    build: .
    ports:
      - "5000:5000"
  redis:
    image: redis

Utilise docker compose up pour lancer l'environnement multi-conteneurs.

📁 Volumes

docker volume create mon_volume

Crée un volume persistant.

docker run -v mon_volume:/data alpine

Monte le volume dans un conteneur.

🌐 Réseaux

docker network create mon_reseau

Crée un réseau personnalisé.

docker run --network mon_reseau ...

Permet aux conteneurs de communiquer.
