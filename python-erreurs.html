Introduction

La gestion des erreurs est essentielle en Python pour rendre vos programmes plus robustes et fiables.

Types d'erreurs

  • SyntaxError : erreur de syntaxe dans le code.
  • ValueError : erreur de type de valeur.
  • TypeError : mauvaise utilisation des types.
  • IndexError : accès à un index hors limites dans une liste.

Utilisation de try / except

try:
    x = int(input("Entrez un nombre : "))
except ValueError:
    print("Ce n'est pas un nombre valide.")
🎁 Bonus : Conseils avancés
  • Utiliser finally pour exécuter du code même après une erreur.
  • Créer ses propres exceptions avec raise.
